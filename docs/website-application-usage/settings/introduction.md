# Introduction

On the Settings page https://dev.worx.id/settings, there are two main sections, namely Tabs and the content of each Tab as shown in the image below.

![](/img/screenshots/website-application-usage/settings/introduction/introduction-1.png)

The content on the "User Details" Tab allows you to change your account detail information. The content on the "Password" Tab allows you to change your password. In the initial condition, the page will display the content in the "User Details" Tab.
